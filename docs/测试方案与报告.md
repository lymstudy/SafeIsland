## 1. 测试框架

```
SafeIsland/
├── rtl/                          RTL 源码 (9 个模块)
│   ├── axi_safety_island_core.v  顶层集成
│   ├── s_axi_config.v            AXI4 Slave 配置 (Shadow/反码/锁)
│   ├── safety_island_core_logic.v 调度器+数据流 (FSM反码/累加保护)
│   ├── safety_island_axi_read_engine.v AXI4 Master 读引擎 (Outstanding/CRC-8)
│   ├── safety_island_fault_detector.v 故障检测 (Stuck阈值/Latent)
│   ├── axi_master.v / axi_slave.v     可复用 AXI4 核心
│   ├── config_checker.v                独立配置校验 (保留)
│   └── axi_safety_island_pkg.vh        公共定义
├── tb/                          测试平台
│   ├── tb_safety_island_top_full.v     22场景集成测试
│   ├── tb_safety_island_fault_injection.v 18场景故障注入
│   ├── tb_config_checker.v             配置校验 (9/9 PASS)
│   └── axi_slave_mem_model.v           AXI Slave BFM
└── tools/                       自动化
    ├── run_tests.py              一键运行
    └── gen_safety_report.py      安全报告生成
```

## 6. 仿真复现

### Icarus Verilog

```powershell
cd SafeIsland
# 顶层集成测试
iverilog -gsystem-verilog -I rtl -I tb -o sim_output/tb_top_full.vvp \
  rtl/axi_master.v rtl/axi_slave.v rtl/s_axi_config.v \
  rtl/safety_island_core_logic.v rtl/safety_island_axi_read_engine.v \
  rtl/safety_island_fault_detector.v rtl/axi_safety_island_core.v \
  tb/axi_slave_mem_model.v tb/tb_safety_island_top_full.v
vvp sim_output/tb_top_full.vvp

# 配置校验
iverilog -gsystem-verilog -I rtl -I tb -o sim_output/tb_config_checker.vvp \
  rtl/config_checker.v tb/tb_config_checker.v
vvp sim_output/tb_config_checker.vvp
```

### ModelSim (完整验证)

```powershell
cd SafeIsland/sim/modelsim
vsim -c -do run_safety_island_top_full_tb.do
vsim -c -do run_safety_island_fault_injection_tb.do
```
